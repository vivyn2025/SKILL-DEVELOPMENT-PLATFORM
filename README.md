# Skill Platform (MVP)

This workspace contains a minimal MVP: Postgres + Express backend and a Vite + React frontend.

Quick start (requires Docker):

```bash
# from repo root
docker compose up --build
```

- Backend: http://localhost:4000/api
- Frontend: http://localhost:5173
